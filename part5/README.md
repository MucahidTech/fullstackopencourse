# Part 5 - Testing React Apps

Testing frontend applications with unit tests, integration tests, and end-to-end testing.

## 🧠 Core Techniques

- 🧪 **Unit & Integration Testing** – Using Vitest and React Testing Library to test components and forms.
- 🌐 **End-to-End Testing** – Using Playwright to simulate real user interactions in a browser.
- 🔐 **Frontend Authentication** – Managing user login state and tokens, and testing authenticated flows.
- 🧭 **Client-Side Routing** – Implementing React Router for navigation and views.
- 💅 **Styling** – Enhancing UI with MaterialUI components.
