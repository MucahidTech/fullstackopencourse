# 📞 Phonebook

A full-stack contact management app that communicates with a backend server. Built through exercises 2.6 - 2.17 of Full Stack Open Course.

## 🎯 What it does

- ➕ Add new contacts (name + number)
- 🔍 Search contacts by name (case-insensitive)
- 🗑️ Delete contacts with confirmation dialog
- ✏️ Update existing contact numbers
- ✅ Shows success/error notifications (auto-dismiss after 5 seconds)

## 🧠 What I practiced

- 📝 Controlled forms with `useState`
- 🌐 Fetching data from server with `axios` + `useEffect`
- 🧹 Extracting API calls into separate service module
- 🔄 HTTP methods: GET, POST, PUT, DELETE
- 🚦 Conditional rendering (filtered list, notifications)
- 💅 Basic Styling with CSS (success/error notifications, basic layout)

## 🚀 How to run

```bash
npm install
npm run dev
```

Also start the JSON server (separate terminal):

```bash
npm run server
```
