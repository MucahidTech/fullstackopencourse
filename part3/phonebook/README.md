# Part 3 - Programming a Server with Node.js and Express

Building a full-stack phonebook application with a RESTful backend, persistent database storage, and automated linting.

## 🧠 Core Techniques

- 🌐 **RESTful API design** – Building routes with Express (GET, POST, PUT, DELETE)
- 🗄️ **Database integration** – Using MongoDB Atlas and Mongoose ODM for persistent storage
- ✅ **Data validation** – Implementing custom validators (phone number format) and built-in validators (min length, required)
- 🛡️ **Error handling** – Centralized error handler middleware with proper HTTP status codes
- 📝 **Logging** – Using Morgan middleware for HTTP request logging
- 🚀 **Full-stack deployment** – Deploying backend and serving frontend build on Render
- 🧪 **Linting** – Configuring ESLint with Prettier to enforce code quality
