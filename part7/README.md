# Part 7 - Advanced State Management & Routing

Deep dive into modern React patterns: custom hooks, React Router, Zustand for client-state, React Query for server-state, Context API, and full-stack integration with a real-world blog application.

## 🧠 Core Techniques

- ⚓ **Custom Hooks** – `useField`, `useAnecdotes` for reusable logic
- 🧭 **React Router** – Routing, navigation, and 404 handling with splat routes
- 🗄️ **Zustand** – Client-state management for notifications, blogs, and user
- ⚛️ **React Query** – Server-state management with `useQuery` and `useMutation`
- 📦 **Context API** – Cross-cutting state (notifications, user) without prop drilling
- 🧩 **Component Architecture** – Reusable, maintainable components with MaterialUI
- 🚨 **Error Boundaries** – Graceful error handling with fallback UI
- 🧹 **Code Quality** – Prettier for automatic code formatting
