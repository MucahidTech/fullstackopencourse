# Part 1 - Introduction to React

React fundamentals through three mini projects.

## 📁 Projects

| Project        | Description                                                                                 |
| -------------- | ------------------------------------------------------------------------------------------- |
| **courseinfo** | Course structure display - components, props, data evolution (variables → objects → arrays) |
| **unicafe**    | Feedback collection (good/neutral/bad) with statistics, conditional rendering, HTML tables  |
| **anecdotes**  | Random anecdote generator with voting system and most-voted display                         |

## 🧠 Core techniques

- React components, props, JSX
- useState hook for state management
- Event handling & conditional rendering
- Immutable state updates (spread operator)
- Derived state & array operations (random index, max value, indexOf)
