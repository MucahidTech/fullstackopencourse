# 📖 Anecdotes

A React app for voting on software engineering anecdotes. Built through exercises 1.12 - 1.14 of Full Stack Open Course.

## 🎯 What it does

- 🎲 Shows random anecdotes with "Next" button
- 🗳️ Vote for your favorite anecdotes
- 🏆 Displays the most voted anecdote

## 🧠 What I practiced

- 🎲 Random selection with `Math.random()` and `Math.floor()`
- 📦 Immutable array updates with spread operator `[...votes]`
- 🔢 Finding max value: `Math.max(...array)` + `indexOf()`

## 🚀 How to run

```bash
npm install
npm run dev
```

Then open http://localhost:5173
